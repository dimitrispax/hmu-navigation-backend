# HMU Navigation API

Backend API for the HMU campus navigation app, serving geospatial data and live class schedules.

Built with Express.js, TypeScript, PostgreSQL, and PostGIS.

Thesis project at [Hellenic Mediterranean University](https://hmu.gr) (2022–2023). Originally hosted on the uni's internal GitLab, it is mirrored here to showcase my progression as a developer and my practical experience with geospatial databases. It is currently archived and no longer actively maintained.
